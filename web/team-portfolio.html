팀원으로 참여하는 방법

  1. members/example.html과 members/example.json을 복사
  2. 두 파일 모두 members/본인이름.html / members/본인이름.json으로 저장
  3. JSON 파일의 내용을 본인 정보로 채우기 (HTML은 자동으로 같은 이름의 JSON을 불러옵니다)
  4. members/members.json 배열에 슬러그 추가 (예: "hong-gildong")
  5. GitHub에 푸시하면 자동으로 이 페이지에 카드가 추가됩니다!
